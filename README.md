# Интернет-Магазин Одежды (PHP версия)

Веб-приложение интернет-магазина одежды для дипломной работы, разработанное на PHP с MySQL базой данных.

## Функциональность

### Для всех пользователей:
- Просмотр каталога товаров с фильтрацией и поиском
- Сортировка товаров по цене и названию
- Просмотр товаров по категориям
- Регистрация и авторизация

### Для покупателей:
- Добавление товаров в корзину
- Оформление заказов
- Просмотр истории заказов
- Управление профилем

### Для магазинов:
- Добавление и управление товарами
- Обработка заказов (подтверждение/отклонение)
- Просмотр статистики продаж

### Для администраторов:
- Управление пользователями
- Создание категорий
- Просмотр всех заказов и товаров
- Полная статистика системы

## Структура файлов

```
php_version/
```

## Установка на OpenServer

1. **Скачайте и установите OpenServer**
   - Перейдите на https://ospanel.io/
   - Скачайте и установите OpenServer

2. **Настройте проект**
   - Скопируйте папку `php_version` в директорию `domains` OpenServer
   - Переименуйте папку в `clothing-store.localhost`

3. **Импортируйте базу данных**
   - Запустите OpenServer
   - Откройте phpMyAdmin (http://localhost/openserver/phpmyadmin/)
   - Создайте новую базу данных `clothing_store`
   - Импортируйте файл `database_php.sql`

4. **Настройте конфигурацию**
   - Откройте файл `config.php`
   - При необходимости измените настройки подключения к базе данных

5. **Запустите приложение**
   - Откройте http://clothing-store.localhost в браузере
